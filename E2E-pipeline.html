End to end pipeline to deploy JAVA maven application

  • Project Description

    This is a project i made as a capstone project for the Professtional certification course in DevOps engineering from Edureka.
    I have created an end to end pipeline to deploy a JAVA maven application using various tools such as jenkins, kubernetes, docker, etc

  • Tools used

    • GIT - for source code management
    • Jenkins - for creating CI/CD Pipeline
    • Docker - for creating the container
    • Kubernetes - for container orchestration
    • Ansible - for configuration management & CD
    • Prometheus & Grafana - for monitoring the application
