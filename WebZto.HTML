Web-ZTO - Zero Touch Operations

  • What is Web-ZTO ?

    web-ZTO is a web version of a TCS ip Named ZTO which was a windows application.
    we made it as a web version so that it could be platform indipendent.

  • What does Web-ZTO do?

    Web-ZTO is a solution that automates various Application / Infra Operations tasks through multimple integrations, automation scripts, automation plugins, etc in Azure Cloud platform thus eliminating the involvement of operational personels and enabling the entire process to be "Zero Touch".
    The Solution consists of integration between ITSM tool JIRA and Azure cloud platform.

  • Technology Stack used in web-ZTO

    • UI
      • React JS
    • Backend
      • Java Springboot
      • Gradle
    • Cloud
      • Azure Cosmos DB
      • Azure DevOps
      • Azure Container Registry
      • Azure Kubernetes service
      • Azure Storage Account
